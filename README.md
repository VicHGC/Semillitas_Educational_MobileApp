# Semillitas_Educational_MobileApp
Aplicación educativa móvil para niños de 3 a 8 años desarrollada con React Native + Expo. Incluye juegos interactivos de matemáticas y español, sistema de monedas y tienda de avatares, dashboard parental con métricas, seguimiento de letras difíciles y autenticación con Google OAuth. Backend en Node.js/Express con MySQL en Digital Ocean.
